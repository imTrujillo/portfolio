👣 Paso 24 - Maqueta del formulario

En esta oportunidad aprenderás a maquetar un formulario con HTML. Además vas a repasar la selección de elementos para dar estilos con CSS y que el formulario quede visualmente acorde al sitio web.

📌 ¡Recuerda mantener las cámaras encendidas en todo el proceso! ¡Sé protagonista de tus objetivos!

 ¡Empecemos! 🚀

✏️ Actividad - Maquetar el formulario

En esta actividad vas a desarrollar el formulario de consultas del portfolio.

Para lograrlo, debes seguir estas instrucciones:

1. Dentro de la carpeta de 📂mi_primera_web, abre el archivo formulario.html.

2. Entre la barra de navegación y el pie de página de la vista, pega el siguiente código html

<section>
      <h2 class="titulo_seccion">Contacto</h2>
      <div class="form_contenedor">
        <div class="container">
          <form>
              <div class="form-group">
                  <label for="nombre">Nombre:</label>
                  <input type="text" id="nombre" name="nombre">
              </div>
              <!-- Agrega más inputs aquí -->
  
             
              <div class="form-group">
                <label for="mensaje">Mensaje:</label>
                <textarea id="mensaje" name="mensaje" rows="4"></textarea>
            </div>
              <button type="submit">Enviar</button>
          </form>
      </div>
</section>

3. Abre con live server el archivo formulario.html y verifica cómo se visualiza en el navegador. 

4. Agrega un nuevo campo que se llame “email”. ¡Atención! El atributo type para email es específico, busca cuál corresponde y colócalo. 

💡El uso de <div class="form-group"> proporciona una estructura semántica al formulario. Cada grupo de elementos relacionados, como un campo de entrada y su etiqueta asociada, se envuelve dentro de este contenedor. Esto hace que el código HTML sea más legible y comprensible tanto para los desarrolladores como para las herramientas de accesibilidad.

Hasta aquí tenemos la estructura del formulario en html, ahora para el CSS, coloca el siguiente código en el archivo formulario.css 

.titulo_seccion {
  position: relative;
  font-size: 40px;
  color: #003750;
  text-align: center;
}
.container {
  max-width: XX; /* Define el ancho máximo del contenedor */
  margin: 0 auto; /* Centra el contenedor horizontalmente */
  padding: XX; /* Agrega un poco de espacio alrededor del formulario */
  border: XX; /* Agrega un borde al contenedor */
  border-radius: XX; /* Agrega bordes redondeados al contenedor */
}
.form-group {
  margin-bottom: 15px; /* Espacio entre cada input */
}
label {
  margin-bottom: 5px;
}
input[type="text"],
input[type="email"],
textarea {
  width: XX; /* Define el ancho máximo del input */
  padding: 8px;
  border: 1px solid #ccc;
  border-radius: 4px;
}
textarea {
  resize: vertical; /* Permite redimensionar verticalmente el textarea */
}
button {
  /* Coloca estilos a tu botón */
}
button:hover {
  background-color: #0056b3;
}

5. Deberás reemplazar las XX por los valores que consideres correspondientes

6. Coloca estilo a tu botón, recuerda usar estilos como width, background-color, border-radius, padding, margin

7. Abre con live server el archivo formulario.html y verifica cómo se ven los estilos

8. Ya tienes el armado inicial del formulario. ¡Modifica los valores que desees para que se adapte a tu diseño! 

💡 Comparte con tus compañeros cómo va quedando tu web, aprovecha ver las dudas con tu equipo o con el mentor.

✅ Checklist

Asegúrate de:

  • Entender cómo maquetar correctamente un formulario HTML.

  • Testea los diferentes tipos de <input> y entender cuándo conviene usar cada uno de ellos.

  • Da un estilo apropiado al formulario para que forme parte de tu sitio web.
